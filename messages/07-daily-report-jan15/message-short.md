## Debug Menu — 18 вариантов секций

Вместо выбора одного "правильного" варианта каждой секции — инфраструктура для переключения между вариантами прямо на сайте. Floating panel в правом нижнем углу, сохраняет выбор в localStorage.

**Grid (фон):** cursor-tracking (cheesy, для сравнения), slow-drift (по умолчанию), static, off. Плюс Achromatic-style направляющие линии с toggle.

**Header:** Два варианта. Pill-navbar (оригинальный Oatmeal) и mega-menu (портированный из Aura). Оба теперь с полными dropdowns.

**Phase 2A mega-menu сделан:** ProductMenu с sidebar и 17 items (было 9), CasesMenu с 11 case studies и метриками, ResourcesMenu с 5 секциями, DocsMenu с SDK grid. Централизованный menuContent.ts на 500 строк.

**TrustedBy:** marquee (для сравнения), static-grid (по умолчанию — как ты показывал на Stripe), static-minimal.

**CoreFeatures:** colorful, muted, monochrome — три уровня контрастности по твоему фидбэку про "буйство цветов".

**Stats:** cards, inline, graph, floating — spring physics, parallax.

**Testimonials:** editorial (крупные цитаты как на Stripe), wall, carousel.

**RoleCards:** cards, tabs, horizontal. Убрал двухлинейные заголовки и декоративные теги.

**Integrations:** static-grid (по умолчанию), marquee, categorized.

Ты сказал "это последний полный ревью, дальше только поблочно" — именно поэтому Debug Menu так важна. Теперь обсуждаем конкретные блоки.

## По твоему видео-фидбэку

**Grid:** Понял правильно при повторном просмотре — сама сетка норм, проблема в следовании за курсором (cheesy). Slow-drift по умолчанию. Плюс добавил направляющие линии как на Attio/Vercel.

**Цвета:** Adapty purple (#6720FF) добавлен в OKLCH. Оливковую палитру сохранил для фонов. Постепенно нахожу места для фиолетового акцента.

**Контраст:** Работаю над "белое на сером вместо серого на сером". Направляющие линии помогают структурировать.

## Находка — Achromatic

Знаю, договорились никаких новых прототипов. Но такого невозможно было пройти мимо — было бы глупо проглядеть.

Achromatic — premium SaaS starter-kit, случайно оказался в паблике на GitHub. Не просто шаблон, а полная инфраструктура: Next.js 16, Auth.js, Prisma, Stripe, Turborepo.

**Три приложения:** Dashboard (auth, billing, settings, teams), Marketing (landing, blog, docs, changelog), Public API. 50+ компонентов в едином стиле. Всё что нам дальше понадобится — уже есть.

Стилистика близка к Oatmeal — минимализм, полированность, современный вкус. Комбинированное использование: Oatmeal для marketing pages, Achromatic для infrastructure (docs, blog, auth, dashboard).

Создал debug menu и там тоже, собрал варианты секций с нашим контентом. Завтра покажу оба на созвоне.
