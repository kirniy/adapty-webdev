### ASCII Wireframes

Для каждой из 14 секций homepage подготовлены ASCII-схемы структуры. Пример секции Hero:

```
┌─────────────────────────────────────────────────────────────┐
│  [Logo]  Features  Pricing  Docs  Blog    [Sign In] [CTA]   │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│          ███████ HEADLINE (2 lines) ███████                 │
│                                                             │
│              Supporting text paragraph                      │
│                                                             │
│           [Primary CTA]  [Secondary CTA]                    │
│                                                             │
│     ┌─────────────────────────────────────────┐             │
│     │                                         │             │
│     │         Hero Visual / Dashboard         │             │
│     │                                         │             │
│     └─────────────────────────────────────────┘             │
│                                                             │
│        Logo1   Logo2   Logo3   Logo4   Logo5                │
└─────────────────────────────────────────────────────────────┘
```

Это позволяет точно воспроизвести структуру во всех 5 вариантах — меняется только styling.

## Ключевые находки

### Универсальные паттерны (используют все 5 сайтов)
- 4px базовая единица spacing
- Геометрические sans-serif шрифты (Inter/Geist)
- Негативный letter-spacing для заголовков
- Blue-purple акцентные цвета
- 6-12px border radius
- 200-300ms длительность анимаций

### Рекомендация для Adapty: DS5 (Hybrid Premium)
- **Тема**: Light с тёплым серым (#F7F7F8) — не чистый белый
- **Шрифт**: Inter (индустриальный стандарт)
- **Анимации**: 150-200ms с bouncy easing
- **Почему**: Широкая аудитория (не только разработчики), приятнее для глаз
