# Апдейт по Phase B: shadcn/ui исследование

Привет, Сергей!

Провёл вечер 13 января за реализацией Phase B прототипов на shadcn/ui. Делюсь результатами и выводами.

---

## TL;DR

**shadcn/ui не подходит для маркетинговых страниц.** Библиотека оптимизирована под dashboards и apps, а не под landing pages с "wow-эффектом". Результаты Phase B значительно уступают Phase A (vanilla Tailwind).

---

## Что было сделано

### Phase A (завершено ранее)
- **adapty-prototype.vercel.app** - 5 дизайн-систем с переключателем
- Чистый Tailwind CSS, без библиотек
- Все 14 секций реализованы
- **Оценка: приемлемо**, есть база для развития

### Phase B Part 1: shadcn/ui (13 января)
Создано 3 прототипа:

| Прототип | Секций | Статус | Оценка |
|----------|--------|--------|--------|
| proto-ds1-linear | 14/14 | Готов структурно | Слабо |
| proto-ds2-attio | 10/14 | Частично | Слабо |
| proto-ds3-polar | 0/14 | Не начат | - |

**Проблемы:**
1. shadcn/ui даёт "generic" look - кнопки, карточки выглядят стандартно
2. Переопределение всех стилей занимает больше времени, чем писать с нуля
3. Даже с Framer Motion анимациями и кастомными иконками (Phosphor) - результат "пластиковый"
4. Библиотека создана для функциональных интерфейсов, не для маркетинга

## Варианты дальнейших действий

### Вариант A: Развивать Phase A (рекомендуется)
- Взять adapty-prototype как базу
- Добавить polish: лучшие анимации, микро-взаимодействия
- Сфокусироваться на одной дизайн-системе (рекомендую DS5 light или DS1 dark)

### Вариант B: Попробовать Tailark
- Tailark - маркетинговые блоки поверх shadcn
- Специально для landing pages
- Может дать лучший результат чем "голый" shadcn

### Вариант C: Hybrid подход
- Маркетинговые секции - vanilla Tailwind (Phase A approach)
- Функциональные компоненты (dropdown, modal) - shadcn
- Best of both worlds

## Учёл твои требования

Напоминаю себе (и тебе) что ты говорил:
- **Тема**: Светлая, менее плотная подача
- **Референсы**: Linear, Attio, Polar, Vercel, Clerk (Stripe с осторожностью)
- **Конкуренты**: RevenueCat, Superwall
- **Примеры желаемого стиля**: apple-ads-manager, apple-fiscal-calendar лендинги

Исходя из этого, рекомендую **DS5 (светлая, теплая)** или **DS2 (Attio-style)** как базу.
