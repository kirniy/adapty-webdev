## Version Naming Convention

- **Major (1.x.x)**: Production release, all features complete
- **Minor (x.1.x)**: New features or significant improvements
- **Patch (x.x.1)**: Bug fixes and minor improvements

### Pre-release Tags
- **alpha**: Early development, core features in progress
- **beta**: Feature complete, undergoing polish and testing
- **rc**: Release candidate, final testing before v1.0
